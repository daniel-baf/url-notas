Proyecto: Sistema de Monitoreo por Audio con IA

Clasificación de sonidos en tiempo real con dispositivos IoT y notificaciones por WhatsApp

Descripción del Proyecto

Este proyecto consiste en un sistema distribuido donde dispositivos ESP32 equipados con micrófonos capturan audio del ambiente y lo envían a un servidor central donde un modelo de inteligencia artificial clasifica los sonidos en diferentes categorías.

Cuando se detecta un sonido de interés (por ejemplo: alarmas, vidrios rompiéndose, gritos), el sistema automáticamente envía una notificación por WhatsApp a los usuarios registrados, incluyendo información sobre el tipo de sonido detectado, la ubicación del dispositivo que lo capturó y una grabación del audio.

El sistema está diseñado para aplicaciones de seguridad y monitoreo ambiental, permitiendo una respuesta rápida ante eventos relevantes.

Dispositivos IoT (ESP32)

  • Captura de audio mediante micrófono integrado
  • Procesamiento inicial (reducción de ruido, normalización)
  • Conexión WiFi para transmisión de datos
  • Envío de archivos de audio al servidor central
  • Manejo de energía eficiente para funcionamiento prolongado

Servidor Central

  • Recepción de archivos de audio desde múltiples dispositivos
  • Clasificación de audio mediante modelo de IA
  • Almacenamiento de registros en base de datos
  • Gestión de usuarios y dispositivos
  • Integración con API de WhatsApp

Sistema de Notificaciones

  • Detección de eventos importantes
  • Generación de mensajes personalizados
  • Envío de alertas por WhatsApp
  • Sistema de prioridades para diferentes tipos de eventos
  • Registro de notificaciones enviadas

Tecnologías Utilizadas
